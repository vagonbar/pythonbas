Curso básico de Python

Este Curso Básico de Python intenta proveer los conocimientos necesarios para programar en el lenguaje Python. El material provisto fue diseñado para ser usado en cursos presenciales, semi presenciales, o para auto estudio. El contenido provee una formación básica en el lenguaje, desde la cual podrá encararse con solvencia el estudio de bibliotecas con propósitos específicos.
