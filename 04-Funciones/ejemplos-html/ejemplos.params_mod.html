ejemplos.params_mod module documentation

Part of ejemplos

Parámetros recibidos por un módulo.

Muestra un mensaje, nombre del módulo y lista de argumentos recibidos. Sugerencias de invocación:

  • python paramsmod
  • python paramsmod "uno" "dos" "tres"
  • python paramsmod.py "uno" 2 2.55 True False, None

La lista de argumentos recibidos queda en argv del módulo sys. El primer elemento de la lista, argv[0], es el nombre del módulo. Observar que todos los argumentos son recibidos como 'str'.
